Цель лабораторной работы

Изучение механизмов авторизации и регистрации при использовании Flask.

Выполненные задачи

  • Установка пакета flask-login
  • Создание модели User на основе UserMixin (id, email, password, name)
  • Реализация страницы регистрации (GET /register)
  • Обработка регистрации (POST /register)
  • Реализация страницы входа (GET /login)
  • Обработка входа (POST /login)
  • Реализация выхода (GET /logout)
  • Защита маршрутов с помощью @login_required
  • Хэширование паролей
